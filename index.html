Progressive visual analytics plays a key role in the analysis of applications that generate large amounts of data in real-time, and one such application is performance analysis on HPC performance data generated from parallel applications running on supercomputers. In general, the collected data is multivariate and of high dimensionality. Furthermore, the high frequency and continuous unbound nature of data streams leads to challenges when deriving knowledge from the underlying observations follows a tri-space data representation, comprising of structural information from network communication, multivariate data collected by profiling the execution, and the temporal context. The complexity of the data combined with the need to analyze in real time demands designing effective visual interactions and visual encodings that help users to draw attention. We report the design and implementation of a visual analytics framework to identify bottlenecks, locate outliers, and form hypotheses about their causes and effects. The system allows the user to effectively explore the tri-space model by coupling appropriate machine learning techniques..... In particular, ...We evaluate the system with use cases to analyze the data collected from the CODES network simulation framework and the ROSS PDES simulator engine.
Visualizing Hierarchical Performance Profiles of Parallel Codes using CallFlow.

Huu Tan Pham Nguyen, Abhinav Bhatele, Nikhil Jain, Suraj P. Kesavan, Harsh Bhatia, Todd Gamblin, Kwan-Liu Ma and, Peer-Timo Bremer.

pdf abstract code
Calling context trees (CCTs) couple performance metrics with call paths, helping understand the execution and performance of parallel programs. To identify performance bottlenecks, programmers and performance analysts visually explore CCTs to form and validate hypotheses regarding degraded performance. However, due to the complexity of parallel programs, existing visual representations do not scale to applications running on a large number of processors. We present CALLFLOW, an interactive visual analysis tool that provides a high-level overview of CCTs together with semantic refinement operations to progressively explore the CCTs. Using a flow-based metaphor, we visualize a CCT by treating execution time as a resource spent during a call chain, and demonstrate the effectiveness of our design with case studies on large-scale, production simulation codes.
A Visual Analytics Framework for Analyzing Parallel and Distributed Computing Applications.

Jianping Kelvin Li, Takanori Fujiwara, Suraj P. Kesavan, , Caitlin Ross, Misbah Mubarak, Christopher D. Carothers, Robert B. Ross, and Kwan-Liu Ma
In Proceedings of Symposium on Visualization in Data Science (VDS) , 2019

pdf abstract code
To optimize the performance and efficiency of HPC applications, programmers and analysts often need to collect various performance metrics for each computer at different time points as well as the communication data between the computers. This results in a complex dataset that consists of multivariate time-series and communication network data, which makes debugging and performance tuning of HPC applications challenging. Automated analytical methods based on statistical analysis and unsupervised learning are often insufficient to support such tasks without the background knowledge from the application programmers. To better explore and analyze a wide spectrum of HPC datasets, effective visual data analytics techniques are needed. In this paper, we present a visual analytics framework for analyzing HPC datasets produced by parallel discrete-event simulations (PDES). Our framework leverages automated time-series analysis methods and effective visualizations to analyze both multivariate time-series and communication network data. Through several case studies for analyzing the performance of PDES, we show that our visual analytics techniques and system can be effective in reasoning multiple performance metrics, temporal behaviors of the simulation, and the communication patterns.
